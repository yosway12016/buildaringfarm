🌦️ Mutation Events

Weather events that can mutate your plants

How Events Work

There are anywhere between 1-3 rolls per event. Every time a roll procs, plants have that respective chance to mutate.

Example: If you have 20 plants down and Black Hole rolls, each one of those 20 plants has a 3% chance to mutate to Void per roll.

Event Event Chance Chance per Roll Multiplier Mutation
🌧️ Rain 40% 8% 1.5x Wet
❄️ Blizzard 20% 4% 1.75x Frozen
🌑 Black Hole 12% 3% 2.25x Void
☢️ Nuclear 7% 2% 3x Radioactive
🌌 Galaxy 4% 1% 5x Rainbow

📊 Rolls Per Event

Each event triggers 1 to 3 rolls. The number of rolls is random, and each roll is independent — meaning a single event could potentially mutate multiple plants, or none at all.

Rain Event starts → 2 rolls trigger
Roll 1: Plant A has 8% chance → Wet!
Roll 2: Plant B has 8% chance → No mutation
